# Changelog

Toutes les modifications notables de ce projet seront documentées dans ce fichier.

Le format est basé sur [Keep a Changelog](https://keepachangelog.com/fr/1.0.0/),
et ce projet adhère au [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-08

### Ajouté
- Infrastructure DevOps complète
- Pipeline CI/CD avec Jenkins
- Monitoring avec Prometheus et Grafana
- Orchestration Kubernetes (K3s)
- GitOps avec ArgoCD
- Analyse de code avec SonarQube
- Gestion Docker avec Portainer
- Application de démonstration Node.js
- Dashboard web
- Scripts d'installation et de déploiement
- Documentation complète
- Support Docker Compose

### Sécurité
- Isolation des containers dans un réseau Docker
- Health checks configurés
- Resource limits définis
- Volumes persistants pour les données

## [0.1.0] - 2026-02-07

### Ajouté
- Configuration initiale du projet
- Structure de base des dossiers
- Premiers manifestes Kubernetes
